# Python API Client

Small Python project for interacting with an HTTP API, handling JSON responses, timeouts, and basic formatting.

## Requirements
- Python 3.9+
- `requests`

## Install
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
## Usage
```bash
python api_client.py http://example.org:3000/
```
